# Little Lemon Restaurant - Front-End Capstone Project

This project is the final capstone for the **Meta Front-End Developer Professional Certificate** on Coursera. It is a responsive React web application for a fictional Mediterranean restaurant called "Little Lemon."

## 🚀 Features

- **Responsive Design:** Optimized for mobile, tablet, and desktop views.
- **Dynamic Booking Form:** Users can reserve a table with real-time available time slots.
- **Form Validation:** Client-side validation to ensure correct data entry.
- **Accessibility:** Implementation of ARIA labels and semantic HTML tags.
- **Unit Testing:** Key components and logic are tested using Jest and React Testing Library.

## 🛠️ Tech Stack

- **React 18**
- **React Router v6** (Navigation)
- **CSS Grid & Flexbox** (Layout)
- **Jest & React Testing Library** (Testing)

## 📦 Project Structure

- `src/components`: Reusable UI elements (Header, Nav, Footer, etc.).
- `src/pages`: Page-level components corresponding to routes.
- `src/utils`: Contains external API logic for booking data.
- `src/__tests__`: Unit tests for form validation and logic.
